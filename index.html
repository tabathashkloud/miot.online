🔐 MIoT: MCP Identity of Things

MIoT is an open protocol for managing machine-auth identities (MAKII) in agentic systems. It defines how secrets, API keys, and AI agents can operate securely under zero-trust conditions using ephemeral KeyTickets and signed credentials (ztsSig).
